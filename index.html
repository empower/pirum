Empower Campaign's PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover empower.github.com/pirum
Listing available packages: 
pear remote-list -c empower
Installing a package: 
pear install empower/package_name
Installing a specific version/stability: 
pear install empower/package_name-1.0.0
pear install empower/package_name-beta
Receiving updates via a feed: 
http://empower.github.com/pirum//feed.xml
